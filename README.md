# AI-Driven Automated Reporting System with LLM APIs

> **A cost-efficient, AI-driven automated reporting system built using free-tier Google Gemini Flash LLM APIs to analyze financial data from Google Sheets and deliver insights automatically to Slack.**

This project demonstrates how real-world AI automation systems can be designed **without paid LLM APIs**, focusing on **cost optimization, scalability, and business usability**.

## ğŸ”„ Workflow

1. **Scheduled Trigger**  
   Runs automatically on a monthly schedule using Zapier.

2. **Google Sheets Data Fetch**  
   Retrieves multiple rows of financial data.

3. **LLM-Based Analysis**  
   Gemini generates insights such as trends, risks, and recommendations based strictly on the provided data.

4. **Slack Notification**  
   A structured financial summary is sent to a Slack channel.
